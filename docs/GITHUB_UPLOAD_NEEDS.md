# GitHub Upload Requirements

This sanitized folder is the only content intended for a public or semi-public
GitHub repository.

## Required From The Owner

1. GitHub username or organization name.
2. Repository name, for example `xrd-embedded-public-demo`.
3. Visibility choice:
   - public, if the contest page must show a public link;
   - private, if the contest accepts a private or invite-only repository;
   - no repository, if the contest field can be filled with `None`.
4. Authentication method:
   - already logged-in browser session that the owner controls; or
   - a fine-grained temporary GitHub credential with only repository content
     read/write permission for the selected repository.

## Not Required

Do not provide account login phrase, recovery code, long-term credential,
private network settings, model service credentials, remote machine
credentials, or any unrelated repository access.

## Upload Boundary

Upload only this folder's sanitized examples, schemas, and documentation.
Do not upload the full project tree, model weights, private datasets, raw
experiment records, deployment scripts, or direct hardware control scripts.
